
# extrator-curriculo-udemy

Extensão simples que extrai todos os títulos e estatísticas de currículo de um curso da Udemy e copia para sua área de transferência.

## Como usar

1. Abra a página de um curso na Udemy (na aba "Conteúdo do curso").
2. Clique no ícone da extensão.
3. Clique em "Extrair currículo"
3. O texto completo será extraído e poderá ser copiado para o clipboard.

Feito!
